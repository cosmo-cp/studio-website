# Cosmo Studio Landing Page

This is the landing page for Cosmo Studio, a universal AI workspace desktop application.

## Features

- **Multi-Provider Support**: Connect to OpenAI, Anthropic, and more.
- **Local Models**: Run local models via Ollama.
- **Unified Interface**: A consistent chat interface for all models.
- **Privacy First**: Local storage and processing.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **UI Components**: Custom components inspired by Shadcn UI
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Fonts**: Figtree, Spectral, IBM Plex Mono

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
